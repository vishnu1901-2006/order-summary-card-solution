## My Process

### Built With

### What I Learned

- **CSS Custom Properties**: Used CSS variables for consistent color management throughout the design
- **Flexbox Mastery**: Applied flexbox for card layout, plan container alignment, and button stacking
- **Responsive Design**: Implemented a mobile-first approach with proper viewport meta tags
- **Hover States**: Added interactive hover effects for buttons and links to improve user experience
- **SVG Background**: Leveraged background images with proper sizing for aesthetic appeal
- **Semantic HTML**: Used proper heading hierarchy and semantic elements for better accessibility
